С гит хаба (внешнего репозитория) на локальный компьютер 
Git clone "сылочка"

Войти в папку с репозиторием 
cd Bug-Reports

Cоздать файл json
touch Bug_Report_1.json

Открыть и заполнить файл 
vim Bug_Report_1.json
"{
        "ID": 1,
        "Title": "В разделе 'Обратная связь' не осуществляется переход по ссылке 'Telegram'",
        "Precondition": "Пользователь не зарегистрирован",
        "Steps": {
                "1.": "Открыть ссайт(ссылка на сайт)",
                "2.": "Клик по Burger menu",
                "3.": "Клик по разделу 'Обратная связь'",
                "4.": "Клик по иконке 'Telegram' в разделе 'Соцсети'"
                        },
        "A.R.": "Переход на страницу с ошибкой 'net::ERR_UNKNOWN_URL_SCHEME'",
        "E.R.": "Отобразилась страница с переходом на соцсеть 'Telegram'",
        "Attachment": "Link",
        "Severity": "Major",
        "Priority": "Medium",
        "Status": "Open",
        "Author": "@yKovalevy",

                "1":{

                        "Browser": "Google Chrome v.108.0.5359.128"
                                }

}"

Сохранить и выйти 
ESC 
:wq
 
Смотрим состояние рабочего каталога
git status

Добавляем файл в промежуточную область 
git add Bug_Report_1.json

Создаем комит
git commit -m "Bug_Reporrt_1"

Загружаем файл на внешний репозиторий 
git push

Изменить статус бага на "Closed"
vim Bug_Report_1.json

Проверить статус 
git status

Добавляем файл в промежуточную область и сразу комитим (т.к. этот файл был уже ранее закомитен и выгружен можем позволить)
git commit -am "status change"

Загружаем файл на внешний репозиторий 
git push
